# SteamOS Dotfiles

My personal SteamOS configuration for my custom steam machine (Blizzard).

Features:

- Package installation
- Zsh configuration
- Ghostty
- Fastfetch
- macOS keyboard shortcuts (xremap)
- OpenRGB (OPTIONAL)
- Ludusavi (OPTIONAL)
- Storage Mounting (OPTIONAL)
- Modular installer

## Installation

```bash
git clone git@github.com:hari8697/dotfiles.git
cd SteamOS
./install.sh
```

## Modules

| Module              | Description               |
| ------------------- | ------------------------- |
| packages            | Pacman & Flatpak packages |
| shell               | Zsh and Ghostty           |
| cli                 | Fastfetch                 |
| apps                | OpenRGB & Ludusavi        |
| storage             | fstab profiles            |
| macos-kbd-shortcuts | xremap configuration      |


See the docs/ directory for details.
